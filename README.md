# 【前端修練精神時光屋】 - Where's Youbike
使用TDX運輸資料流通服務，以及UI設計師投稿的設計製作的全台Youbike腳踏車地圖。


## 網站功能
- 使用者可搜尋自己附近的Youbike站，並查看可借車輛及可環空位。
- 使用者可依照縣市搜尋腳踏車道，並查看車道路線。
- 使用者可查看自己附近的景點及美食。

## 環境建置與需求

- vue: v3.0.0
